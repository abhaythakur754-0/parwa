AGENT_COMMS.md — Weekly Communication Template
Cleared and archived at the end of every week. All 7 agents read and write here. 
How to Use This File 
Every agent reads this file at the start of their session. Every agent writes only to their own sections. The Manager writes tasks and advice. Builders write their status. The Tester writes verification reports. The Assistance Agent writes the founder's summary. This file is the team's shared brain for one week. 
 
  WEEKLY RESET: At the end of every week, the Manager archives this file to docs/weekly_logs/week_N_comms.md
  and replaces it with this blank template. New agents start fresh each week. PROJECT_STATE.md carries the memory forward — this file
   carries the
   conversation.
